## MonitorChain Developer Sandbox tools
NodeJS AccessLibrary, monitor reset script and integration code snippets.

## Install

- Clone the repo
- Edit the config file (config/default.js)
- Add a list of tokens to config/tokens.js

## Usage

```bash
node reset.js
```

For more information see the [monitorchain-interface-library](https://github.com/ZenchainSoftware/monitorchain-interface-library).
